# Student Average (CMD)

Simple Python script that calculates the average of entered marks.

## Features
- Reads 3 (or more) marks from user input.
- Prints the average.
- Includes a pause so the console doesn't close immediately.

## How to run
1. Install Python 3.
2. Run:
```bash
python student_average.py
